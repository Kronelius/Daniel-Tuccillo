# Images

Drop a real social-share image here named `og-image.png` (recommended size
**1200×630**). It's what shows when your link is shared on LinkedIn, iMessage,
etc. Until then the `og:image` tag points to a file that doesn't exist yet,
which is harmless.

Good options for this image: your headshot + tagline, or a simple branded
card with "Automation for Home Service Businesses."
